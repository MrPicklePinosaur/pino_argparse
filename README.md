# pino_argparse
a tiny zero-dependency argparsing library

</div>

## USING IN YOUR PROJECT

add the follow to your `Cargo.toml`:
```
pino_argparse = { git = "https://github.com/MrPicklePinosaur/pino_argparse" }
```
